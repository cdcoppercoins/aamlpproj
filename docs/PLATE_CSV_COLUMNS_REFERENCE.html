Plate CSV Columns Reference

MiniLicensePlates.com — Catalog pricing guide. Each row is a plate type (or variety) with values for six conditions (MT, EX, VG, G, FR, PO). Many plates have no photos; those with photos may have an optional back image. Export as CSV when ready for database import.

Column headers (copy into row 1)

set_code,set_name,cat_ref,company,image_base,image_ext,has_back_image,jurisdiction,jurisdiction_type,year,serial_number,width_inches,height_inches,value_mt,value_ex,value_vg,value_g,value_fr,value_po,variety_key,variety_notes,state_embossed,legend_embossed,notes,sort_order

Column definitions

Column Notes
set_codeFolder code, e.g. m88p, c36g
set_nameDisplay name, e.g. 1988 Post Cereal Plates
cat_refCatalog reference number (max 10 chars)
companyCompany that issued/manufactured the plate (e.g. Post, General Mills, Topps)
image_baseBase filename without _a/_b suffix, e.g. AL, ON. Leave blank for plates with no photo.
image_extImage extension: jpg, jpeg, png, gif, etc. Leave blank when no photo. Back image ({base}_b.{ext}) is optional when front exists.
has_back_image1 = back image exists, 0 = front only, blank = unknown
jurisdictionRegion/country name: Alabama, Ontario, Mexico, etc.
jurisdiction_typeOne of: us_state, ca_province, ca_territory, foreign_country
yearNumeric year, e.g. 1988
serial_numberNumbers or letters displayed on the plate itself
width_inchesWidth in inches (e.g. 0.0625, 1.625). Up to 4 decimal places (thousandths).
height_inchesHeight in inches (e.g. 0.0625, 1.625). Up to 4 decimal places (thousandths).
value_mtValue for Mint condition. Use number or range (e.g. 3.50 or 3-5). Leave blank if N/A.
value_exValue for Excellent condition. Leave blank if N/A.
value_vgValue for Very Good condition. Leave blank if N/A.
value_gValue for Good condition. Leave blank if N/A.
value_frValue for Fair condition. Leave blank if N/A.
value_poValue for Poor condition. Leave blank if N/A.
variety_keyShort ID for variety: "base" or blank for base plate; "4h", "6e", etc. for varieties. Needed to distinguish multiple rows per plate.
variety_notesFree-text description of variety differences (e.g. "4-hole embossed", "6-hole flat", "large font").
state_embossed1 = state name on plate is embossed, 0 = no, blank = unknown
legend_embossed1 = legend/slogan on plate is embossed, 0 = no, blank = unknown
notesOptional notes or variety info
sort_orderInteger for display order within set (0, 1, 2…)
