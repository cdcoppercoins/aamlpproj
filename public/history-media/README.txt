History timeline images
=======================

Put one image per timeline entry here. Match the filename to config/history_timeline.php
(for example: goudey-cards.jpg).

IMPORTANT: This folder is named history-media (not "history") because a folder
named public/history breaks the /history web page when using php artisan serve.

Recommended: JPG or WebP, roughly 800px wide or larger, landscape when possible.

If an entry has no image, set 'image' => null in the config file.
